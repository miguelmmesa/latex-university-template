% PURPOSE

% Main entry point of the LaTeX project.
% This file orchestrates the compilation of the entire document.
%
% It loads:
%   - Global configuration
%   - Required packages
%   - Document style modules
%   - Acronym definitions
%   - Frontmatter elements
%   - Chapters
%   - Bibliography
%   - Appendices
%
%
% ARCHITECTURE ROLE
% ---------------------------------
% This file acts as the central controller of the document build process.
%

%
%   00_config/
%   07_styles/
%
%
% DEPENDENCIES
% ------------
%
% Configuration
%   00_config/config.tex
%   00_config/packages.tex
%
% Styles
%   07_styles/*
%
% Frontmatter
%   01_frontmatter/*
%
% Content
%   02_chapters/*
%
% Bibliography
%   05_bibliography/*
%
% Appendices
%   06_appendices/*
%
%
% NOTES FOR DEVELOPERS
% --------------------
%
% Do NOT place document content directly in this file.
%
% Content should always be placed in:
%   02_chapters/
%
% This file should only orchestrate the structure of the document.
%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
