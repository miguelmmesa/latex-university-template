\chapter{Introducción}

Esta plantilla ha sido diseñada para facilitar la redacción de trabajos académicos
como informes técnicos, proyectos universitarios o Trabajos Fin de Grado.

El objetivo principal es separar claramente cuatro elementos del proyecto:

\begin{itemize}
\item Configuración del documento
\item Estilos tipográficos
\item Contenido académico
\item Recursos gráficos y bibliográficos
\end{itemize}

Gracias a esta separación, el autor puede concentrarse exclusivamente en escribir
el contenido sin preocuparse por el formato del documento.

\section{Características de la plantilla}

La plantilla incluye las siguientes funcionalidades:

\begin{itemize}
\item Portada institucional configurable con índice automático
\item Lista de figuras, tablas, códigos y algoritmos
\item Gestión automática de acrónimos y bibliografía con estilo APA
\item Estilos tipográficos profesionales
\item Entornos para código fuente y algoritmos
\end{itemize}

\section{Filosofía de uso}

La plantilla está diseñada siguiendo una filosofía simple:

\begin{itemize}
\item El contenido se escribe en la carpeta \texttt{02\_chapters}
\item Las figuras se guardan en \texttt{03\_figures}
\item Las tablas se almacenan en \texttt{04\_tables}
\item Las referencias bibliográficas se almacenan en \texttt{05\_bibliography}
\end{itemize}

Esta organización permite mantener el proyecto limpio, modular y fácil de mantener.
